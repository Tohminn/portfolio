About This Project

Downloads

This is one method to do secure file downloads. A POST request is made to a generic 'download.php' address with the file name supplied. After authenticating the request, the file contents are retrieved, base64 encoded, and returned to the client in JSON object format. Then a little bit of DOM manipulation occurs to trigger the file download dialog pop up. This process also allows files to be generated dynamically by the server (think reports) and be returned along with additional data to the client. A downside is that large files or slow internet connection can make it appear as if nothing is happening. This can be circumvented by using event callbacks to update a loading bar.
